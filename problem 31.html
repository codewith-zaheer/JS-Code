Problem 31

Write a JavaScript function called getWeightStatus that takes a BMI value as an argument and returns a string indicating the weight status based on the BMI value. Use the following ranges: • BMI less than 18.5: "Underweight" • BMI between 18.5 and 24.9: "Normal weight" • BMI between 25 and 29.9: "Overweight" • BMI 30 or greater: "Obese".
